# SafeHer – Women Safety System using Raspberry Pi, GSM, GPS, Pulse & Sound Sensors

## Overview
SafeHer is an IoT-based women safety system developed using Raspberry Pi, GSM, GPS, Pulse Sensor, and Sound Sensor. The system is designed to provide immediate assistance during emergencies by monitoring the user's condition and enabling quick communication with emergency contacts.

## Features
- One-touch SOS emergency alert
- Real-time GPS location tracking
- Automatic SMS notification using GSM
- Pulse rate monitoring
- Sound detection for emergency situations
- Raspberry Pi-based embedded system

## Components Used
- Raspberry Pi
- GSM Module
- GPS Module
- Pulse Sensor
- Sound Sensor
- Push Button
- Power Supply

## Working
1. The user presses the panic button or an emergency condition is detected.
2. The Raspberry Pi reads the user's GPS location and sensor data.
3. The GSM module sends an SOS message with the current location to predefined emergency contacts.
4. Pulse and sound sensor readings can assist in identifying critical situations and support a faster emergency response.
